# Avance 1 - Gestión de Inventarios

Este proyecto corresponde al primer avance de una aplicación de gestión de inventarios para una ferretería. El sistema permite administrar productos mediante una lista enlazada simple, utilizando Java y una interfaz de consola.

También se agregó una interfaz gráfica desarrollada con Java Swing como complemento visual para facilitar el registro, búsqueda, modificación y visualización de productos.

## Funcionalidades

* Insertar productos al inicio y al final de la lista.
* Mostrar productos registrados.
* Buscar productos por nombre.
* Modificar datos de un producto.
* Agregar imágenes a un producto.
* Eliminar productos.
* Generar un reporte de costos por producto.
* Calcular el costo total acumulado.

## Clases principales

* `Producto.java`: almacena los datos de cada producto.
* `NodoProducto.java`: representa el nodo de la lista enlazada.
* `ListaProductos.java`: contiene las operaciones de la lista enlazada simple.
* `Main.java`: contiene el menú de consola.
* `VentanaProductos.java`: contiene la interfaz gráfica opcional.

## Ejecución

Para ejecutar la versión de consola:

```text
Main.java
```

Para ejecutar la interfaz gráfica:

```text
VentanaProductos.java
```

## Tecnologías utilizadas

* Java
* Java Swing
* IntelliJ IDEA
* Lista enlazada simple
* ArrayList para rutas de imágenes
